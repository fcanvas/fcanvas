fcanvas-next

A next version library fcanvas, Its syntax looks like Konva.js but it uses ES6's Proxy response system and minimizes redrawing. View docs

Installation

NPM / Yarn:

yarn add fcanvas-next

CDN:

<script src="https://unpkg.com/fcanvas-next"></script>
